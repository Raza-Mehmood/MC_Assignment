How To Add A File In GitHub Repository

Before following these steps do make sure git () is installed in your PC

  • First Create an GitHub Account
  • Create an respository in GitHub Account
  • Copy the repsitory URL in code section
  • Now Create an html file in VS Code
  • Now open Terminal in VS Code
  • Now enter following commands:
    • git init
    • git add "my_file"
    • git commit -m "Making first commit"
    • git remote add origin *copied URL of respository*
    • git push -u master
